
Example:  Moving Representations

This example shows how to move custom vector data points over time.
10,000 dynamic entities are created and moved randomly every frame.
Also demonstrates some simple keyboard navigation.

Requires data from the Geoweb3d DC sample project that can be downloaded here:
http://www.geoweb3d.com/download/sample-projects/
Change the 'root_path' variable to the installed location of the DC project.
